Se inicia a investigar sobre cómo transferir (Stream) datos de una Raspberry Pi a una computadora. Las opciones encontradas fueron:

	Uso de TCP/IP para la comunicación. Implica Programa en la RPi y también en el computador receptor (que "bombee" a la RPi).

	Uso de Initial State para recolectar información de IoT y centralizarla, para luego de alguna forma extraerla para análisis. 

Se pidió prestado al Depto. de Computación una Raspberry Pi para el propósito.
